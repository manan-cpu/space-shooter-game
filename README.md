# Space Shooter Game
A Space Shooter game built with Pygame. 

## How to Play
- Move the ship using `W`, `A`, `S`, `D` keys.
- Shoot bullets with the `Space` key.
- Avoid asteroids and enemy bullets to survive!

## Setup
Install Pygame:
```bash
pip install pygame
